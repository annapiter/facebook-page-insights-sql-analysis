# Facebook Page Insights SQL Analysis

Business-oriented SQL analysis of Facebook audience, engagement, demographics, and growth metrics using SQLite.

## Objectives
Analyze:
- audience reach
- engagement trends
- geographic fan distribution
- demographic splits
- posting performance
- month-over-month growth

## Skills Demonstrated
- SQL aggregations
- CTEs
- Window functions (LAG)
- Joins
- Date/time analysis
- Percentage calculations
- Business KPI analysis

## Key Analyses
- Top countries by fans
- Penetration ratio analysis
- Audience split by age and gender
- English-speaking audience analysis
- Best posting day and time
- Month-over-month engagement deltas

## Tools
- SQLite
- SQLiteStudio
- GitHub
