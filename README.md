# Facebook Page Insights SQL Analysis

Business-oriented SQL analysis of Facebook audience, engagement, demographics, and growth metrics using SQLite.

## Project Documentation

* 📄 **Business Questions and Results:** [business_questions.pdf](business_questions.pdf)
* 📊 **Project Presentation:** [sql_project_presentation.pdf](sql_project_presentation.pdf)

## Project Objectives

* Analyze audience demographics and geographic distribution
* Identify growth opportunities across countries and cities
* Evaluate language preferences and potential buying power
* Understand engagement patterns by day and time
* Measure month-over-month changes in engagement metrics
* Translate SQL outputs into business insights and recommendations

## Features

* Audience and demographic analysis
* Location and penetration analysis
* Language and buying power analysis
* Engagement analysis by day of week and time of day
* Month-over-month trend analysis
* Additional exploratory business analyses
* Multiple solution approaches using CTEs, subqueries, and window functions

## SQL Concepts Used

* Aggregations (`SUM`, `AVG`, `COUNT`)
* `GROUP BY` and `ORDER BY`
* `JOIN`s
* Common Table Expressions (CTEs)
* Window Functions (`LAG`, `OVER`)
* Subqueries
* Date and Time Analysis with `strftime`
* KPI Calculations
